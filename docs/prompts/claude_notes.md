# Claude コマンド実行メモ

- 実行バイナリ: `/home/dev/.local/bin/claude`
- インタラクティブに続けたい場合は `claude` を実行し、1問目→回答確認→2問目…と順に入力するだけで同じセッションが継続される。
- 非対話モードで1問ずつ実行したい場合は `claude --print "質問"` を使う。直前のセッションを引き継ぎたい場合は `claude --continue --print "質問"` を併用する。
- この環境では CLI が外部ディレクトリに設定ファイルを作成しようとするため、Codex から呼び出す際は「with_escalated_permissions=true」を付けて実行する（権限エラーを防ぐため）。
- 途中のログを明示したい場合は、擬似TTY経由で `claude` を起動し標準出力をキャプチャする方法もあるが、基本的には上記フローで十分。
- 最初に作業者レベルのプロンプト(/home/dev/backend/Orchestrator/prompts/worker.md)を読み込ませること。
- 指示が通らない場合は新しく会話を始めてもよいです。
- 分割して出力させるときは続けてくださいと伝えること。
